# UrlBuddy

The purpose of this project was to write a small Google Chrome extension in pure javascript to help easily see and remove URL parameters.

## Installing URL Buddy

This Google Chrome extension can be found [Here](https://chrome.google.com/webstore/detail/url-buddy/aodnklgahfdobaojpniffbjockilkndp) to install directly.
